# Prompt Library

Approved AI prompts for bank staff, stored as plain markdown files in git.

- **Folders are categories.** Adding a category = adding a folder.
- Every prompt is a `.md` file: YAML front-matter (metadata), then the prompt body.
- `_templates/` holds authoring templates and is not shown in the Prompt Library app.
- Changes to `main` require an approved review — suggest edits through the
  Prompt Library app, or open a pull request here directly.

Browse these prompts in the Prompt Library app rather than in Gitea — it
renders the metadata, hides deprecated prompts, and copies the prompt text
without the front-matter.
